# Docker Node
Docker Image for building Node Apps with node-gyp and Python. (e.g. with GitlabCI)
## Versions
- Alpine 3.8
- Node 10.7 / NPM 6.2
- Git
- node-gyp
- Python
## Image
You can find the Docker Image on Docker Hub: [smlabt/node](https://hub.docker.com/r/smlabt/node/)
```
docker pull smlabt/node
```
